# Import dat

Návod pro import dat do ISMD nástroje z různých zdrojů.

## Podporované formáty

### CSV soubory

- Kódování: UTF-8, Windows-1250
- Oddělovače: čárka, středník, tabulátor
- Možnost nastavení hlaviček

### Excel soubory

- Formáty: .xlsx, .xls
- Více listů v jednom souboru
- Automatická detekce datových typů

### JSON a XML

- Strukturovaná data
- Vnořené objekty
- Pole a kolekce

## Postup importu

1. **Výběr souboru**
   - Klikněte na "Import dat"
   - Vyberte soubor z počítače

2. **Nastavení parametrů**
   - Zvolte formát
   - Nastavte kódování
   - Definujte oddělovače

3. **Náhled a validace**
   - Zkontrolujte náhled dat
   - Ověřte datové typy
