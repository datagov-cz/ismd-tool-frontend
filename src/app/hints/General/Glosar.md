# Slovník pojmů ISMD

Přehled nejčastěji používaných termínů v systému.

## A–O

**IRI** (Internationalized Resource Identifier)
Globálně unikátní identifikátor zdroje, rozšíření URI podporující unicode znaky.

**JSON-LD**
Formát pro linked data založený na JSON. Přidává `@context` pro mapování klíčů na IRI.

**Koncept / Pojem**
Základní stavební blok ontologie – třída, vlastnost nebo hodnota.

**NKD** (Národní katalog dat)
Veřejný registr datových sad, ontologií a číselníků spravovaných státní správou.

**Ontologie**
Formální popis znalostní domény pomocí tříd, vlastností a vztahů.

## O–Z

**OWL** (Web Ontology Language)
W3C standard pro reprezentaci ontologií, staví na RDF a RDFS.

**Prefix**
Krátká zkratka nahrazující opakující se část IRI, např. `foaf:` místo `http://xmlns.com/foaf/0.1/`.

**RDF** (Resource Description Framework)
W3C standard pro popis zdrojů na webu ve formě trojic subjekt–predikát–objekt.

**RDFS** (RDF Schema)
Rozšíření RDF o základní ontologické konstrukty jako `rdfs:Class` nebo `rdfs:subClassOf`.

**Turtle (TTL)**
Kompaktní textový formát pro zápis RDF dat, čitelnější než RDF/XML.
