## Typy analýz

### Popisné statistiky
- Průměr, medián, modus
- Směrodatná odchylka
- Kvartily a percentily

### Inferenční statistiky
- Testování hypotéz
- Intervaly spolehlivosti
- Korelační analýza

### Prediktivní analýzy
- Regresní modely
- Časové řady
- Klasifikace
