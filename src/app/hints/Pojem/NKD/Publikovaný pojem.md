# Publikovaný pojem v NKD

Tato stránka zobrazuje pojem publikovaný v Národním katalogu dat. Jde o autoritativní, veřejně dostupnou verzi konceptu.

## Přístup přes IRI

Publikovaný pojem je identifikován přes parametr `iri` v URL:

```
/concept/nkd?iri=https://slovník.gov.cz/základní/pojem/fyzická-osoba
```

## Co je zobrazeno

- Kompletní metadata pojmu ve stavu ke dni publikace
- Všechny vztahy k ostatním pojmům ve stejné ontologii
- Mapování na pojmy z jiných ontologií (pokud existuje)
- Časová osa verzí – kdy byl pojem naposledy změněn

## Trvalý odkaz

Každý publikovaný pojem má trvalý odkaz (permalink):

```
https://slovník.gov.cz/{ontologie}/pojem/{název}
```

Tento odkaz je vhodné používat v dokumentaci a propojení s jinými systémy.

## Chci pojem upravit

Publikované pojmy nelze přímo editovat. Pro navržení změny:

1. Přihlaste se do systému
2. Klikněte na **Navrhnout změnu**
3. Vyplňte důvod změny a nová data
4. Návrh bude odeslán správci příslušné ontologie
