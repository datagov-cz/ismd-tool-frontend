# Nastavení

Tato sekce obsahuje návody pro konfiguraci ISMD nástroje.

## Obsah sekce

- [Uživatelské nastavení](./uzivatelske-nastaveni.md) - Personalizace rozhraní
- [Systémové nastavení](./systemove-nastaveni.md) - Konfigurace systému

## Typy nastavení

### Uživatelské preference
- Jazyk rozhraní
- Barevné téma
- Formáty dat a času
- Výchozí hodnoty

### Systémová konfigurace
- Připojení k databázím
- Síťové nastavení
- Bezpečnostní politiky
- Výkonnostní parametry

## Správa nastavení

### Import/Export
- Zálohování konfigurace
- Sdílení nastavení mezi uživateli
- Obnovení výchozích hodnot

### Profily
- Různé profily pro různé úkoly
- Rychlé přepínání mezi profily
- Správa více konfigurací
